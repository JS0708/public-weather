# 🌤️ Weather Map Forecast Studio

FastAPI와 Streamlit을 활용한 전국 중기예보 시각화 프로젝트입니다.

## ✨ 주요 기능
- **Interactive Map:** 전국 권역별 예보 상태를 색상으로 시각화
- **Timeline Autoplay:** 날짜별 예보 변화를 애니메이션으로 확인
- **Modern UI:** Glassmorphism 디자인 적용 및 다크 모드 지원
